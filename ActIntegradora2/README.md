# Act Integradora 2

Programa en Python que procesa la entrada de la situación problema 2 y entrega, en consola:

- Árbol de expansión mínima calculado con Kruskal para definir el cableado óptimo.
- Ruta del agente de correspondencia resolviendo el TSP con Held-Karp (o heurística para N grandes).
- Flujo máximo entre la primera y la última colonia usando Edmonds-Karp.
- Polígonos de Voronoi que asignan cada punto nuevo a la central más cercana.

Las matrices se leen del archivo indicado al ejecutar `python main.py entrada.txt`, y todos los resultados se imprimen siguiendo el formato solicitado.
